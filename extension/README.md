# Cafe MD Browser Extension

Chrome/Edge 浏览器扩展，支持右键打开 Markdown 文件。

## 功能

- 点击扩展图标打开 Cafe MD
- 右键菜单：选中文本后右键可「Open with Cafe MD」
- 支持拖拽 .md 文件到浏览器打开

## 免费安装方法（开发者模式）

### Chrome 浏览器

1. 下载扩展文件
   - 从 GitHub 下载 `extension` 文件夹
   
2. 打开扩展管理页面
   - 在地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

3. 开启开发者模式
   - 右上角开关打开「开发者模式」

4. 加载扩展
   - 点击左上角「加载已解压的扩展程序」
   - 选择 `extension` 文件夹

5. 完成！
   - 扩展图标会出现在工具栏
   - 右键菜单中会出现「Open with Cafe MD」

### Edge 浏览器

1. 打开扩展管理页面
   - 在地址栏输入：`edge://extensions/`

2. 开启开发人员模式
   - 左侧开关打开「开发人员模式」

3. 加载扩展
   - 点击「加载解压缩的扩展」
   - 选择 `extension` 文件夹

4. 完成！

## 准备图标文件

扩展需要图标文件才能正常显示。请准备以下尺寸的图标：

- `icons/icon16.png` - 16x16 像素
- `icons/icon48.png` - 48x48 像素  
- `icons/icon128.png` - 128x128 像素

### 生成图标的方法

1. 使用在线工具：
   - https://favicon.io/
   - https://realfavicongenerator.net/

2. 或使用现有图片调整大小

3. 将生成的图标放入 `extension/icons/` 文件夹

## 使用方法

### 方法一：点击图标
- 点击浏览器工具栏中的 Cafe MD 图标
- 自动打开新标签页进入编辑器

### 方法二：右键菜单
- 在网页中选中任意文本
- 右键 → 「Open with Cafe MD」
- 选中的内容会在编辑器中打开

### 方法三：拖拽文件
- 将 .md 文件拖拽到浏览器窗口
- 自动在 Cafe MD 中打开

## 故障排除

### 扩展无法加载
- 确保已开启「开发者模式」
- 确保选择了正确的 `extension` 文件夹
- 检查图标文件是否存在

### 右键菜单不显示
- 刷新页面后重试
- 检查扩展是否已启用

### 图标显示异常
- 确保 `icons` 文件夹中有正确尺寸的图标文件
