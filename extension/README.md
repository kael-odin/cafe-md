# Cafe MD Browser Extension

Chrome/Edge 浏览器扩展，支持右键打开 Markdown 文件。

## 安装方法

### 开发者模式安装（无需审核）

1. 打开 Chrome/Edge 浏览器
2. 访问 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `extension` 文件夹

### Chrome Web Store 上架（需要审核）

1. 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/dev/dashboard)
2. 支付一次性 $5 美元注册费
3. 点击「添加新项目」
4. 上传扩展的 ZIP 压缩包
5. 填写商店信息（截图、描述等）
6. 提交审核（通常 1-3 个工作日）

## 功能

- 点击扩展图标打开 Cafe MD
- 右键菜单：选中文本后右键可「Open with Cafe MD」
- 支持拖拽 .md 文件到浏览器打开

## 创建图标

需要准备以下尺寸的图标：
- `icons/icon16.png` - 16x16
- `icons/icon48.png` - 48x48  
- `icons/icon128.png` - 128x128

可以使用在线工具生成：
- https://favicon.io/
- https://realfavicongenerator.net/
